1) Extract ZIP

2) Install all dependencies:
   npm run install:all

3) Start backend:
   cd backend
   copy .env.example .env
   npm run dev

4) Start frontend in new terminal:
   cd frontend
   npm run dev

5) Open:
   http://localhost:3000

Login:
   manager / manager123

NOTE:
This upgraded version has offline demo fallback. If backend/MongoDB is not running, manager login still opens the dashboard.
For full database + AI features, add MONGODB_URI and GROQ_API_KEY in backend/.env.

ELEVENLABS SETUP:
1. Open backend/.env
2. Add ELEVENLABS_API_KEY and ELEVENLABS_VOICE_ID
3. Restart backend server
4. Login as manager/admin
5. Turn on FRIDAY and test voice reply
