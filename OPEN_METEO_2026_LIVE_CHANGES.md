# 2026 Open-Meteo Live Data Update

- City Basic Data now shows 2026 Live instead of 2025.
- Weather, temperature, AQI, wind and rain are fetched from Open-Meteo.
- 2021-2025 historical data remains constant.
- 2026 is the only live-changing layer.
- Selected Map Area no longer shows any AI recommendation automatically.
- AI remains idle until the user asks a question or clicks an analysis button.
- API source panel added in the dashboard.
- Report labels updated to 2026 Live.
- Progress bar overflow protection added.
